Myoelectric Prosthetic Hand

A bionic hand controlled by muscle signals

Figure 1: The completed prosthetic hand showing its mechanical structure and electronic components.

Project Overview

This project involved designing and building a myoelectric prosthetic hand that can be controlled using muscle signals from the user's residual limb. The hand features individual finger control, multiple grip patterns, and force feedback. The system uses surface EMG sensors to detect muscle activity and translates these signals into precise hand movements.

Figure 2: Exploded view of the hand's mechanical components showing the tendon-driven finger actuation system.

Custom 3D-printed components and tendon-driven actuation system.

Control System

EMG signal processing and motor control implementation.

Technical Details

The prosthetic hand was built using a combination of 3D printing, custom electronics, and machine learning. The system uses EMG sensors to detect muscle activity, processes these signals using a microcontroller, and controls servo motors to actuate the fingers. The design includes force sensors for grip strength feedback and a modular architecture that allows for easy maintenance and upgrades.
