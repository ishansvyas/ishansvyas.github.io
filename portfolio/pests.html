A Pest Detection and Deterrence Sensor and Actuation System for Organic Farms

IoT-Based Smart Agriculture System Using Multi-Level Detection and Humane Deterrence

Summary

Organic farms face significant challenges from pest damage, with limited options for protection compared to conventional farms that can use pesticides. Our team developed an intelligent IoT-based solution that detects mid-size farm pests such as raccoons, groundhogs, squirrels, and rats, then activates humane deterrence mechanisms.

The system features a two-stage detection pipeline: low-power PIR sensors provide 360-degree motion detection, which triggers high-level analysis using cameras and a custom-trained YOLOv8 machine learning model for species identification. When a pest is confirmed, the system wirelessly activates a multi-modal deterrence subsystem featuring sound (predator calls and high-pitched noises), lights (LED flood lights), and motion (a motorized zipline scarecrow moving over a 15-foot span).

Our system successfully achieved 86% detection accuracy, operates on solar power with a 12V 25AH LiFePO4 battery, and is built with NOP-compliant materials suitable for organic farms.

My Contribution

I focused primarily on the deterrence subsystem while also managing project coordination.

First Half of Semester - Conceptual Design:

  • Led conceptual design of the deterrence system using Pugh scoring matrices to evaluate alternatives
  • Developed high-level designs for motion-based zipline scarecrow, sound-based speaker system, and floodlight-based light deterrent
  • Ensured all deterrence options were compatible with single solar-panel power supply
  • Created detailed Gantt chart to organize project timeline and maintain steady progress

Second Half of Semester - Sound Deterrent Implementation:

  • Designed and built the sound deterrent subsystem
  • Developed circuit diagram and performed power analysis
  • Selected and ordered components, tested and amplified speaker system
  • Stored sound files on SD card and integrated WiFi triggering
  • Soldered and ruggedized circuit using Deutsch connectors
  • 3D-printed waterproof mounting system for field deployment

System Integration:

  • Programmed main deterrence logic on Arduino, integrating all subsystem code
  • Implemented non-blocking timing structures to enable simultaneous triggering of all deterrents

Project Management:

  • Managed coordination for major presentations and reports
  • Organized contributions from all eight team members and assigned sections
  • Ensured compliance with content and formatting requirements
