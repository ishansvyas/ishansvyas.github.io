100 MHz CPU in Verilog

A custom MIPS processor implementation

Figure 1: Block diagram of the MIPS processor showing major components and data paths.

Project Overview

This project involved designing and implementing a fully functional MIPS processor in Verilog, capable of running at 100 MHz. The processor features a 5-stage pipeline, hazard detection, and forwarding units. It was implemented on an FPGA and successfully ran various test programs, demonstrating the ability to execute complex instructions and handle data hazards efficiently.

Technical Details

The processor was implemented using Verilog HDL and synthesized for a Xilinx FPGA. Key features include a 32-bit architecture, support for the MIPS instruction set, branch prediction, and memory management. The design was verified using both simulation and hardware testing, achieving the target clock frequency of 100 MHz while maintaining correct operation.
