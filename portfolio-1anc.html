Ancestry Estimator

A machine learning approach to predicting genetic ancestry

Figure 1: Visualization of genetic haplogroups and their geographic distribution across different populations.

Project Overview

This project combined bioinformatics, genetics, and computer programming to analyze ancestral DNA and trace patterns of human migration. The goal was to develop a custom software tool that could interpret raw genetic data and map lineage-based mutations to geographic regions over time.

Figure 2: Principal component analysis (PCA) plot showing genetic clustering of different population groups.

Technical Focus

The project was implemented in R, a statistical programming language widely used in bioinformatics. After an initial phase of learning the language and its data handling capabilities, a custom program was written in four modular stages:

  • Data Import: Load genetic data and population reference databases.
  • Data Matching: Align and merge mutation data across multiple sources.
  • Mutation Analysis: Identify relevant single nucleotide polymorphisms (SNPs) associated with known haplogroups.
  • Output Generation: Print inferred ancestral paths and likely geographic origins.

Figure 3: Flowchart of the ancestry prediction algorithm showing data processing and classification steps.

Biological Context

The program focused on Y-DNA and mtDNA mutations, which are passed down paternally and maternally, respectively. By tracing these mutations and comparing them to published reference datasets, the software could generate rough maps of ancestral migration routes and origin points. This applied both historical population data and modern genomics to a real-world lineage analysis.
